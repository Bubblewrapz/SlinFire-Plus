RELEASE 1.3 (Patch 21 Feb 2022)

Thank you for trying out Doxy's Modified m0re, based on the original hud by m0re and supported by Hypnootize.

ORIGINAL DOWNLOAD: https://huds.tf/site/s-m0re-Hud
----------------------------------------------------------------------------

This HUD was created with the combination of six huds (so far): m0rehud (duh.); m0rehud black; Improved Default Hud; FlawHud; broeselhud and budhud.

Doxy's Modified m0re was created for the purpose of keeping the HUD simple for competitive play, but also suitable for casual players to use and enjoy.

----------------------------------------------------------------------------
FEATURES SO FAR: (COMPARED TO STOCK M0RE)

- An improved and simple uber percentage counter under the crosshair, made so that it stands out from the background gameplay, without taking too much space. (Inspired by Mr. Slin's HUD)

- A customized uber counter for all those 6s medics out there, now the number will go from 0 to 100 in 60 seconds (instead of 55 seconds in stock), to make it usable for lower division players.

- New main font.

- Viaduct background from FlawHud to make the menu buttons stand out.

(also I kept the customizations folder if you prefer the classic look of m0re than my modifications.)

-----------------------------------------------------------------------------
COMPONENTS:

- Default Hud + Improved Default Hud: Match status panel.

- arrekhud: Advanced spectator hud in comp matches.

- m0rehud: The base of the modifications, uber percentage and uber counter.

- m0rehud black: Class selection menu and Scoreboard.

- FlawHud: Menu Background.

- budhud: Health and ammo font.

- broeselhud: Winpanel.
